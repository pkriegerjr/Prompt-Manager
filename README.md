Prompt-Manager: Ars Promptum
Ars Promptum é uma plataforma centralizada projetada para desenvolvedores e copywriters que buscam elevar o nível de sua interação com IAs. O sistema permite salvar, categorizar e compartilhar os melhores prompts de forma estruturada, transformando experimentação em ativos reutilizáveis.

         Objetivo
Oferecer uma ferramenta de gerenciamento que suporte desde a criação simples até o compartilhamento seguro e a busca avançada por intenção.

         Funcionalidades Principais
Gerenciamento de Prompts (CRUD) Criação Dinâmica: Suporte a títulos, descrições e corpo do prompt com interpolação de variáveis através da sintaxe {{variavel}}.

Controle de Versão: Registro automático de data de modificação e autor em cada edição.

Segurança de Dados: Sistema de exclusão com confirmação e lixeira com retenção de 30 dias.

Metadados: Visualização completa com contador de uso e tags associadas.

Organização e Busca Tags Inteligentes: Sistema de múltiplas tags com autocomplete e gerenciamento centralizado.

Busca Semântica: Encontre prompts por intenção usando linguagem natural, com ranking por relevância, popularidade ou data.

Filtros Avançados: Lógica combinatória (AND) para refinar buscas por categorias e tags.

Experiência do Usuário (UX) Cópia Inteligente: Botão de cópia direta com modal integrado para preenchimento de variáveis dinâmicas.

Favoritos: Sistema de toggle para salvar e acessar rapidamente seus prompts preferidos em uma visão dedicada.

Colaboração e Visibilidade Níveis de Acesso: Controle se o seu prompt é Privado, para o seu Time, ou Público.

Compartilhamento Externo: Geração de links públicos revogáveis (apenas leitura) para facilitar a colaboração fora da plataforma.
